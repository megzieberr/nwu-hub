# NWU Study Hub

A private, single-user study dashboard for my university modules — upcoming deadlines,
study goals, a study log, and interactive summaries made by my per-module tutors.

**Stack:** React + Vite + Tailwind, Supabase backend, deployed to GitHub Pages via Actions.

All personal data lives in Supabase behind a login (row-level security). This repo is only
the app shell — no course material is stored here.
